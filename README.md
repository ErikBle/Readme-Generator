# Readme-Generator

## General Info:
A node.js app that generates readme files from the cmd line.

## Programs/Languages used:
* Html
* CSS
* Javascript
* Node.js

## Instructions:
Use "npm install" to get the proper packages.
<br>
Then run "node index.js" to start the app.
